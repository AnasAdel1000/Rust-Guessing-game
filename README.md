**Guessing Game 🦀**:
A classic number-guessing game built with Rust as part of a learning journey into systems programming.

**🎮 How to Play**:
The program generates a random number between 1 and 100.

You input your guess via the terminal.

The program will tell you if your guess is too high or too low.

If you guess the correct number, you win, and the game ends!

**🧠 Concepts Learned**:
Variables & Mutability: Understanding let vs. let mut.

Error Handling: Using match instead of expect for robust, crash-free execution.

Control Flow: Using loop for continuous gameplay.

External Crates: Using the rand crate for random number generation.

Type Conversion: Converting user-input String to u32 using parse().

**🛠️ Built With**:
Rust: The core programming language,
Cargo: Rust's built-in package manager and build system.

**🚀 How to run**:
Just play "program.exe" :)
